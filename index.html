It didn't take me long to realize that code tutorials can only help you so much. The best way to learn to code is to simply start coding and learn as you go. Here's some of the various projects I've worked on on my own and in school.
Censure
I worked on Censure on a team with 3 other students for a class last spring. Censure is a web application that connects to the Facebook, Twitter, and Tumblr APIs and uses them to grab all of a user's posts and run them through a censoring algorithm. There's an adjustable difficulty, and users can view all flagged posts and delete them directly from the app. It basically checks your social media for "bad posts".
Censure was a lot of code, we learned a lot about connecting to APIs, and structuring a web application. If I could go back and rewrite Censure it would be a whole lot cleaner, but it took a lot of time from the 4 of us to finish before the end of the class.
